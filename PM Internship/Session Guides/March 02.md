## 3. Budget Tracking Made Simple (15 minutes)

## 3.1 The Traffic Light System

text
GREEN: Under budget (room for extras)
YELLOW: On budget (watch closely)
RED: Over budget (action needed NOW)

**Weekly Check:**

text
Week 7: 73% spent vs 65% planned = RED
Why? SMS costs 2x expected
Fix? Reduce SMS testing volume

## 3.2 Cost Categories to Watch

**High Risk:**

1. **Cloud costs** - scale up = bill jumps 3x
    
2. **Third-party APIs** - Twilio, Razorpay (usage-based)
    
3. **Overtime hours** - "crunch time" kills budget
    
4. **Scope creep** - new features = new people/tools
    

**Low Risk:**

- Jira licenses (fixed)
    
- Basic design tools (fixed)

## 4. Cost Alerts & Action Triggers (10 minutes)

## 4.1 When to Panic

text
Alert Level 1 (Yellow): Any category >110% planned
Alert Level 2 (Red): Total spend >85% before 80% work done
Alert Level 3 (EMERGENCY): Any week >15% budget burn

**Schedula Alert Week 7:**

text
Twilio: 180% of planned → Alert Level 1
Total spend: 73% vs 65% work → Alert Level 2

## 4.2 3 Ways to Save Money

text
1. DEFER: "WhatsApp to Phase 2" (most common)
2. REDUCE: "SMS reminders to 5000/month max"
3. NEGOTIATE: "Twilio volume discount for startups"
